#CashTracker

This web application is built using Django for the backend and React with Tailwind CSS for the frontend. It provides functionality for basic financial calculations and budget management.

## Features

- **Budget Management**: Create, edit, and delete budgets.
- **Expense Tracking**: Track and manage your expenses.
- **Financial Calculations**: Perform basic financial calculations like total expenses, remaining budget, etc.

## Getting Started

### Prerequisites

Make sure you have the following installed on your machine:
- [Python](https://www.python.org/)
- [Node.js](https://nodejs.org/)

### Setting Up the Backend (Django)

1. Navigate to the `backend` directory: `cd backend/`.
2. Create a virtual environment: `python -m venv venv`.
3. Activate the virtual environment:
   - On Windows: `venv\Scripts\activate`
   - On macOS and Linux: `source venv/bin/activate`
4. Install the required packages: `pip install -r requirements.txt`.
5. Run the Django server: `python manage.py runserver`.

### Setting Up the Frontend (React with Tailwind CSS)

1. Navigate to the `frontend` directory: `cd frontend/`.
2. Install dependencies: `npm install`.
3. Start the React development server: `npm start`.

## Usage

1. Access the application at the specified URL after starting both the backend and frontend.
2. Create budgets, manage expenses, and perform financial calculations using the user interface.
